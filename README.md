# Unity UGUI 登录注册系统 Demo

## 📖 项目简介
一个完整的登录注册界面系统 Demo，包含用户登录、注册、服务器选择、提示弹窗等模块。该项目旨在实践 UGUI 的界面搭建、UI 事件处理、面板管理、数据持久化以及简单的动画过渡效果。

## 🛠️ 技术栈
- Unity 2021 LTS
- UGUI
- C#
- JSON 数据存储
- 单例模式
- 资源管理（Resources 加载）

## ✨ 核心功能
- **登录模块**：支持输入验证、记住密码、自动登录功能。
- **注册模块**：新用户注册，用户名唯一性校验，注册成功后自动填充登录信息。
- **服务器选择**：模拟服务器列表展示，支持按页切换、服务器状态图标显示（火爆、流畅、维护等），选择后记录首选服务器。
- **提示弹窗**：统一的提示框，用于显示错误或成功信息。
- **面板管理**：通过 UIManager 统一管理 UI 面板的显示、隐藏和销毁，支持淡入淡出动画。
- **数据持久化**：使用 JSON 存储登录数据、注册用户信息和服务器列表，实现数据本地化。

## 💡 实现要点
- 所有面板继承自 BasePanel，统一处理淡入淡出动画和生命周期。
- UIManager 采用单例模式，负责面板的创建、缓存和销毁，支持通过泛型方法获取面板实例。
- LoginManager 作为全局数据管理器，处理用户数据验证、注册和登录状态保存。
- 服务器列表通过 Resources.Load 动态加载图标（SpriteAtlas）和预制体，实现 UI 复用。
- 通过 Toggle 联动实现记住密码与自动登录的逻辑绑定。

## 🚀 如何运行
1. 克隆仓库到本地。
2. 使用 Unity 打开项目（建议版本 2020.3 及以上）。
3. 打开场景 LoginScene。
4. 运行场景即可体验登录注册流程。
